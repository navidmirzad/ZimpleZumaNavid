Zimple Zuma Navid

This is a console based game:

The following is how it works:

- You will get a list of random colored balls

- Your goal is to link 3 or more of one colored balls next to each other

- This will trigger an event which then removes the colored balls... Thats it.

Functions you can call on the linkedlist:

- linkedlist.insertBallAfterIndex(index, nodeData)
